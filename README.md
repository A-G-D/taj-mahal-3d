# TAJ MAHAL 3D MODEL

## Project Setup

Install dependencies:

```sh
sudo apt install openscad -y --no-install-recommends
```

Clone the repository:

```sh
git clone https://github.com/A-G-D/taj-majal-3d.git
cd taj-mahal-3d
```

To build the STL file:

```sh
openscad -o taj_mahal.stl src/main.scad
```
